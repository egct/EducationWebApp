Introducción

OpenGL (Open Graphics Library) es una especifiación estándar para desarrollar gráficos por computadora en 2D y 3D. Este estándar define una API multilenguaje y multiplataforma cuya interfaz consiste en más de 250 funciones diferentes que pueden utilizarse para dibujar escenas bidimensionales básicas y escenas tridimensionales complejas a partir de primitivas geómetricas 2D tales como puntos, líneas y triángulos (Woo, M., et al., 2005).


Figura 5. Capítulo III.

El API y las librerías de OpenGL fue desarrollado por la empresa Silicon Graphics, Inc. (SGI), en 1992 y se lo utiliza en aplicaciones de CAD, realidad virtual, realidad aumentada, representación científica, visualización de información, simuladores de carros, simuladores de vuelo, entre otras aplicaciones (ver Figura 5.1). Además, OpenGL es ampliamente utilizado en el desarrollo de videojuegos donde compite con Direct3D de la empresa Microsoft para plataformas de Microsoft Windows. (Shreiner, D., 2005). La empresa Silicon Graphics, Inc. (SGI), fue fundada en el año de 1982 por el Dr. James H. Clark quien dejó de trabajar como profesor asociado de Ingeniería Eléctrica en la Universidad de Stanford para formar SGI junto con Abbey Silverstone y luego se unieron un grupo de estudiantes graduados de Stanford como: Kurt Akeley, Tom Davis, Rocky Rhodes, Mark Hannah, Herb Kuta, y Mark Grossman. La empresa Mayfield Group proveyó la financiación inicial de Silicon Graphics, Inc. (SGI). Esta empresa fue comprada en 2009 por Rackable Systems y en año 2016, Hewlett Packard Enterprise (HPE) completó la adquisición de SGI (Kahle, B., Gilliat, B., 2001), (Silicon Graphics-Wikipedia, 2021).


Figura 5.1. Computadoras de Silicon Graphics en el año de 1997 (Vidani, P., 2021).

La biblioteca Open Toolkit mejor conocida como OpenTK, es una biblioteca de gráficos de C# creada por Stefanos Apostolopoulos (2015), bajo la licencia de MIT License. Esta biblioteca reemplaza al Tao Framework creada por Randy Ridge (2008), bajo la licencia de MIT License, la cual también fue una biblioteca de C# que brindó acceso a bibliotecas de juegos y gráficos por computadora con OpenGL y SDL (Tao Framework-Wikipedia, 2021).

OpenTK, brinda acceso a herramientas gráficas contenidas en OpenGL, OpenCL y OpenAL, con una variedad de lenguajes basados en CLR como C# y F# entre otros. En particular, se puede usar con cualquier lenguaje que utilice .NET Framework o Mono Framework para versiones anteriores a 4.0 y .NET Core 3.1 y posteriores, continuando con .NET 5, para la versión principal, cuyos programas pueden ejecutarse en Visual Studio en versiones como 2017, 2019 y 2022 (OpenTK-Wikipedia, 2021).

En este capítulo, se discute la definición de puntos, líneas, triángulos, cuadriláteros, y primitivas en 3D básicas como cubos y conos, así como también sus relaciones para construir gráficos por computadora utilizando OpenTK con OpenGL y el lenguaje de programación C#, sobre el framework .NET con Visual Studio 2019.

5.1. OpenTK

OpenTK (Open Toolkit), es un conjunto de APIs gráficas desarrolladas para C#. Estas APIs son portátiles en varios sistemas operativos (OS) y rápidas en el momento de ejecutarse. OpenTK, tiene un conjunto de clases y librerías de bajo nivel para OpenGL, OpenGL ES, OpenAL y OpenCL, ya que parte del core (núcleo) de OpenTK está construido en C/C++. OpenTK, se ejecuta en todas las plataformas principales como Windows, Linux y MAC. Además, potencia el desarrollo de cientos de aplicaciones gráficas, juegos en 2D y 3D y programas de investigación matemática y científica (De Vries, J., Perks, D., 2021).

OpenTK proporciona varias bibliotecas de utilidades, incluido un paquete de matemáticas y álgebra lineal, un sistema de ventanas y manejo de entrada de datos. OpenTK no es un motor de juegos, ni intenta serlo: ofrece un conjunto útil de herramientas y permite decidir cómo usarlas en diferentes tipos de aplicaciones. Además, es el punto de partida ideal si se desea escribir un juego o un motor de juegos y también una aplicación científica desde cero (De Vries, J., Perks, D., 2021).

De acuerdo con (De Vries, J., Perks, D., 2021), las principales APIs de OpenTK son:

  • OpenTK.Audio.OpenAL
  • OpenTK.Compute.Native
  • OpenTK.Compute.OpenCL
  • OpenTK.Core
  • OpenTK.Graphics
  • OpenTK.Input.Hid
  • OpenTK.Mathematics
  • OpenTK.Platform.Windows

5.1.1. OpenGL, OpenCL y OpenAL

OpenGL es una biblioteca de representación de gráficos por computadora multiplataforma, desarrollada originalmente por Silicon Graphics (SGI) y ahora mantenida por el grupo Khronos. OpenGL se usa para todo, desde videojuegos hasta herramientas CAD, navegadores web y teléfonos móviles.

OpenCL es una especificación de computación paralela multiplataforma, desarrollada originalmente por Apple y ahora mantenida por el grupo Khronos. Se utiliza principalmente para realizar cálculos en GPU que, de otro modo, se calcularían de forma menos eficiente en CPU, pero también se puede utilizar para controlar procesadores de señales digitales (DSP), matrices de puertas programables en campo (FPGA) y otros tipos de aceleradores de cálculo de hardware.

OpenAL es una biblioteca de audio 3D multiplataforma, desarrollada originalmente por Loki Software y lanzada como una especificación abierta por Creative Labs, con múltiples implementaciones tanto en hardware como en software. OpenAL se usa en muchos juegos para proporcionar efectos de sonido y música y es el sucesor de EAX y A3D.

  • OpenTK, brinda acceso a herramientas gráficas contenidas en OpenGL, OpenCL y OpenAL, con una variedad de lenguajes basados en CLR como C# y F# entre otros.
  • OpenTK proporciona varias bibliotecas de utilidades, incluido un paquete de matemáticas y álgebra lineal, un sistema de ventanas y manejo de entrada de datos.
  • Las principales APIs de OpenTK son: a) OpenTK.Audio.OpenAL; b) OpenTK.Compute.Native; c) OpenTK.Compute.OpenCL; d) OpenTK.Core; e) OpenTK.Graphics; f) OpenTK.Input.Hid; g) OpenTK.Mathematics; h) OpenTK.Platform.Windows.

  • OpenAL es una biblioteca de audio 3D multiplataforma, desarrollada originalmente por Loki Software y lanzada como una especificación abierta por Creative Labs, con múltiples implementaciones tanto en hardware como en software.
  • La clase GameWindow contiene métodos multiplataforma para crear y renderizar en una ventana OpenGL, manejar la entrada y cargar recursos.
